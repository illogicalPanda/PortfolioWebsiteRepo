Adventure 1

This project was developed over two months period with a main focus on structured environment development with aspects of collaborative design while using minimal interactive elements such as objects and UI. During the first month, I planned the environment through documentaion and real-world visual references before blocking out the environment. In the second month, I refined the layouts, implemented detailed assets, and expanded the environment while integrating my work with peers, requiring consitent designs and seamless transitions. This project demonstrates skills in collabortion, planning, and the ablity to adapt one's work into a unified experience with others.

Team Adventure Project Showcase
  • Tools Used: Unreal Engine, Unreal Landscapes, Unreal Marketplace, PureRef
  • Created a walkable, playable environment set in a summer setting, divided into structed sections to guider user exploration and pacing
  • Designed level progression from starter areas such as a camp and forest pathwaay into mid-level areas such as an abandoned barn, cave, and ravine leading into the end-game section including an old factory, run-down jail, and forgotten cabin.
  • Built the initial environment using a blockmesh/blockout approach to establish the layout flow. improve environment usability, and improve player experience before final environment detailing.
  • Expanded the environment using Unreal Engine landscape tools to create a more natural and immersive world.
  • Integrated two additional peer-create levels using Unreal Engine Level Streaming and refined the beginning and end of my level to allow for a consitent flow and seamless transition between works.
